# Blog LLM Readiness

Primary source:
- `/PLAYBOOK/blog-llm-readiness-spec.md`

Every canonical GEO-Pulse article should:
- answer the core question early
- include direct-answer blocks
- use concrete headings
- use stable terminology
- support internal linking
- remain easy to extract and cite

Fail the draft if it is:
- fluffy
- overlong without structure
- all persuasion and no explanation
- too vague to quote accurately
