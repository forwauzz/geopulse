# Editorial Rules

Use these rules for every draft:

## Must do

- lead with the point
- separate evidence from interpretation
- keep product mention subordinate to usefulness
- prefer practical diagnosis over trend commentary
- respect the current `free scan` CTA

## Must not do

- imply ranking guarantees
- imply citation guarantees
- invent benchmark authority
- turn weak anecdotal evidence into proof
- write like a generic AI content generator

## Research discipline

- use social-research packets for pain-point discovery and language
- do not use them as standalone proof for hard public claims without separate verification
- if a statistic is not verified independently, treat it as internal direction only
